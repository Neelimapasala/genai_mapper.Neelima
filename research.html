{% if sec.name == 'Abstract' %}

Strengthen the problem framing

{% elif sec.name == 'Literature Review' %}
Group references into themes
Avoid listing papers one by one — group them by approach, dataset, or technique for better flow.
Add more recent papers (last 3 years)

{% elif sec.name == 'Methodology' %}

{% elif sec.name == 'Results' %}
Add more visualisations
